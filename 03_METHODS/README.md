# 03_METHODS

Conductivity measurement protocols, machining test procedures, and metrics definitions.

## Contents

| File | Description |
|------|-------------|
| `conductivity_protocol.md` | 4-probe DC and eddy current measurement procedures |
| `machining_protocol.md` | Turning test setup per ISO 3685 |
| `metrics_definitions.md` | Definitions of VB, Ra, Fc, tool life, %IACS |

## Conventions

- All conductivity values normalised to %IACS or S/m with temperature noted
- Machinability metrics follow ISO 3685 definitions where applicable
- Measurement uncertainty reported per ASTM B193 / E1004
